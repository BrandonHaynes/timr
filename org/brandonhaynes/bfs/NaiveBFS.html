NaiveBFS

class NaiveBFS extends BFSTool

Hadoop Tool for executing a naive BFS over a graph

Sample command line: hadoop jar SS-BFS.jar -input hdfs:///myGraph -D localOutputPath=file:///aLocalPath -D verbose=true -D steps=3 -D algorithm=naive
